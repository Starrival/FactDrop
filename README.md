# FactDrop
Discover something new every time you visit. Click the button to reveal a random fact that's fun, fascinating, and worth sharing.


> A simple website built entirely with HTML.

## About

This project is **not intended to showcase advanced web development skills**. Instead, it demonstrates my ability to learn a new technology quickly and apply it to build something complete.

Before starting this project, I had no experience with HTML. I spent **8 days learning the fundamentals** and built this website with the help of online documentation and AI assistance. My objective was to prove that I can rapidly acquire new skills and turn them into a working project.

I believe that the ability to learn efficiently is just as important as the technologies one already knows.

## Features

- Built using pure HTML
- No frameworks or libraries
- Lightweight and fast
- Structured, semantic markup
